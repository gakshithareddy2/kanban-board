# TaskFlow - Kanban Board

A modern Kanban board web app to manage tasks efficiently with priority levels and progress tracking.

---

## 🚀 Features

- Add tasks with priority (Low / Medium / High)
- Move tasks (To Do → In Progress → Done)
- Delete tasks
- Real-time notifications
- Visual progress tracking with pie chart
- Clean and responsive UI

---

## 🛠 Tech Stack

- React (Vite)
- Tailwind CSS
- Recharts
- React Toastify
